⚡ What are Bolt-On Sidecar Adapters?

In GRIM, Bolt-On Adapters allow attaching and detaching fine-tuned LoRA weights directly into model files in-place using the backup2 residual slot without rewriting or duplicating base model weights.

Why Bolt-On Adapters?

  • Zero Model Duplication: Swap specialized domain adapters (coding, medical, roleplay) instantly onto a single base model file.
  • Reversible Detach: Detaching a bolt-on adapter zero-fills the backup2 residual slot without altering file structure or metadata shape.
  • Sidecar File Format: Training outputs are saved as lightweight .grim.train sidecar files containing adapter matrices and AdamW moments.
