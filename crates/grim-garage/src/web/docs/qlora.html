📖 Beginner's Guide to QLoRA Fine-Tuning

QLoRA (Quantized Low-Rank Adaptation) is a revolutionary fine-tuning technique that enables training massive foundation models (like Qwen, Nemotron, GLM, or Gemini) on standard consumer GPUs with minimal memory usage.

Key Concepts

  • Frozen 4-Bit Base Model: The base model weights remain 100% frozen on disk or memory in 4-bit NormalFloat (NF4) format. Base weights are never updated during backpropagation.
  • Trainable Adapter Matrices (A & B): Low-rank decomposition matrices ($W = W_0 + \frac{\alpha}{r} B \cdot A$) are injected alongside attention and MLP linear projections. Only parameters in $A$ and $B$ receive gradients.
  • Zero-Copy Memory Stream: GRIM streams base weights lazily from disk per transformer block, running just-in-time dequantization directly inside ROCm HIP registers.

Memory Comparison

Full 16-bit fine-tuning of an 8B model requires over 64 GB VRAM. QLoRA reduces peak memory footprint to under 6 GB VRAM without degrading fine-tuning accuracy.
