🎮 AMD ROCm Hardware Acceleration Explained

GRIM is built from the ground up to leverage AMD ROCm HIP (Heterogeneous-computing Interface for Portability) for high-throughput GPU inference and QLoRA fine-tuning.

Architecture Features

  • WMMA & MFMA Acceleration: Employs Wave Matrix Multiply Accumulate (RDNA3) and Matrix Fused Multiply Add (CDNA) hardware matrix cores for GEMM operations.
  • Fused Dequantization GEMM Kernels: grim_fused_dequant_backward_gemm_f16 dequantizes weights on-the-fly inside GPU vector registers, eliminating unnecessary VRAM allocations.
  • Wavefront Execution: Supports Wave32 (RDNA architecture) and Wave64 (CDNA architecture) warp execution profiles.
