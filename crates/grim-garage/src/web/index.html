Training Architecture Thesis

Unsloth-Parity Zero-Copy Memory Stream:

  • Base weights remain frozen and read lazily per-tensor from disk.
  • Dequantization runs just-in-time inside ROCm fused kernels.
  • Only trainable adapter parameters (A & B) + AdamW moments live in VRAM.
  • Gradients accumulate into .grim.train sidecar format.

ℹ️ Understanding Hardware Acceleration in GRIM

Hardware Backend Delineation:

  • AMD ROCm HIP Backend: Native fused dequantization kernels run on AMD Radeon (RDNA) and Instinct (CDNA) GPUs.
  • NVIDIA CUDA Backend: NVIDIA GPUs execute via CUDA kernels / Vulkan backend without ROCm HIP emulation.
  • Automatic Fallback: Systems without discrete GPUs run multi-threaded SIMD CPU kernels.
